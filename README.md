# bike.sales-ved
# 🚴‍♂ Bike Sales Power BI Project

This Power BI project analyzes bike sales data to understand customer purchasing patterns based on age, gender, income, and region.  
The dashboard includes interactive visuals and KPIs such as total customers, total bikes sold, and purchase percentage.  
It helps identify key sales trends and supports better marketing and business decisions.
